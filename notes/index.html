Research Notes

Protocol research, HID decoding references, device assessments, and raw data captures gathered during MockTab development. These are working notes, included as supporting evidence for the implementation choices in the source code.

Protocol reference

  • HID Specs Reference — transport layer, descriptors, and init sequences
  • HID Family Reference — all registry devices by parser family
  • HID Decoding Research — Art Pen rotation packet decoding across USB and BLE
  • HID Experimental Reference — IntuosV2 layouts not yet verified against live hardware
  • Input Decode Reference — packet structures and field decoding, 2003–2017 devices
  • Protocol Specification Notes — design spec for a user-mode macOS driver

Bluetooth & wireless

  • HID Bluetooth Research — BLE HOGP report structures for IntuosV2 and WL devices
  • Bluetooth Battery Reference — battery byte offsets and decode tables by kernel type
  • Wireless Specification Notes — WAK dongle encapsulation and proprietary RF formats

Device-specific

  • PTH-660 Report Layout Reference — full 192-byte USB report layout for Intuos Pro M
  • PTH-850K Reference — Korean SKU; identical protocol to the global PTH-850
  • PTH-860 Reference — Intuos Pro L gen2 USB and BT, including the dual-Bluetooth identity quirk
  • Post-2020 Tablets — Preliminary Research — families and report layouts for newer devices
  • LED Reference — wacom_led_control() dispatch and per-family feature reports

Pen features

  • Art Pen Rotation Notes — how barrel rotation is encoded and scaled to 0–360°
  • Art Pen Rotation Observations — raw capture log
  • HID Rotation Reference — how Krita, Photoshop, and Rebelle interpret macOS NSEvent rotation
  • HID Touch Reference — capacitive and strip touch support across Cintiq and Intuos
  • Parallax Compensation Reference — affine and perspective correction for Cintiq calibration

Linux driver comparison

  • Linux HID Report Reference — verbatim wacom_features table from the kernel
